# Code-Quiz
# test your code knowledge 
Once the start button is clicked, a timer will start to count down.
There are 90 secs alotted for the whole quiz.
A series of mulitple choice questions that change dynamically will display.
There are fifteen seconds per question, with a total of 6 questions.
If the user gets a question wrong then 15 seconds are deducted from remaing time.
After the quiz the user is presented with a box to record score.
Other features inclued, a half second delay to display if correct or wrong and an option to record current score to compare to others.
